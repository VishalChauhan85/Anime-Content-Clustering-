# Frontend — Anime Cluster Predictor (React + Vite)

## Local
```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

## Deploy to Vercel
1. Import the GitHub repo on Vercel.
2. Root directory: `frontend`
3. Framework preset: **Vite** (auto-detected).
4. Environment variable: `VITE_API_URL=https://<your-render-service>.onrender.com`
